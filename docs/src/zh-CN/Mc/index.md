# Minecraft 工具

Minecraft 资源处理和翻译的专业工具套件。

## 可用工具

### JSON 翻译器
用于 Minecraft 语言文件和资源包的高级 JSON 文件翻译工具。

[打开 JSON 翻译器](./JsonTranslator.md)

## 功能特性

- JSON 结构保持
- 批量翻译支持
- 多语言资源包生成
- 语法验证
- 键值对编辑
- 导出为多种格式

## 使用场景

- 翻译 Minecraft 模组
- 创建多语言资源包
- 管理语言文件
- JSON 格式间转换
