# Fate/Grand Order 工具

FGO 从者立绘提取和处理的综合工具套件。

## 可用工具

### 角色查找器
具有精确选择控制和纵横比预设的从者立绘提取工具。

[打开角色查找器](./CharaFinder.md)

## 功能特性

- 高分辨率从者立绘提取
- 自定义纵横比选择（1:1、4:3、3:2、16:9、自定义）
- 像素级精确选择控制
- 背景颜色自定义
- 批量处理支持
- 自动裁剪的人脸检测
- 多语言从者数据库

## 技术详情

该工具与 Atlas Academy API 集成，为所有 FGO 服务器（NA、JP、CN）提供准确的从者数据，支持各种用途的高效立绘提取。
