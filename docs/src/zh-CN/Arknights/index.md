# 明日方舟工具

专业的明日方舟游戏资源提取和分析工具套件。

## 可用工具

### 角色查找器
支持多变体和图像裁剪功能的高级角色立绘提取工具。

[打开角色查找器](./CharaFinder.md)

### 剧情追踪器
通过基于名称的搜索和剧情阅读器集成，追踪角色在游戏中的剧情出场记录。

[打开剧情追踪器](./StoryTracker.md)

## 功能特性

- 多语言支持（中文、英文、日文）
- 角色变体检测和管理
- 剧情出场追踪
- 与外部剧情阅读器集成
- 批量处理功能
- 自动立绘裁剪的人脸检测

## 技术详情

该工具套件从官方明日方舟游戏仓库处理数据，并为资源提取和分析提供直观的界面。
