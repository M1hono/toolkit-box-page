# Hero Configuration

Hero runtime frontmatter for home page composition.

## Pages

- [Hero Runtime](./heroRuntime) - Complete hero frontmatter contract with examples
- [All Hero Config](../../hero/AllConfig) - Full API reference table for every field

## Key Sections

- **Background**: `hero.background` — `image | video | color | shader | particles`, plus multi-layer support
- **Waves**: `hero.waves` — SVG wave bridge with per-layer control
- **Image**: `hero.image` — `image | video | gif | model3d | lottie` with frame and glow options
- **Typography**: `hero.typography` — motion profiles (`floating-tilt`, `grouped-float`, `slanted-wrap`)
- **Colors**: `hero.colors` — text colors, nav bar colors, and search box colors
- **Floating**: `hero.floating` — `text | card | image | lottie | badge | icon | stat | code | shape`
- **Actions**: `hero.actions` — buttons with theme, linkKey, and style overrides
