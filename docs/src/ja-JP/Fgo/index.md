# Fate/Grand Order ツール

FGO サーヴァント立ち絵検索とカード生成のページ一覧です。

## ページ

### サーヴァント立ち絵検索
サーヴァント記録を検索し、素材バリアントを切り替え、切り抜き範囲を指定して画像を出力します。

[サーヴァント立ち絵検索を開く](./CharaFinder.md)

### FGO カード生成
アップロード画像、テキスト、アイコン重ね合わせ、数値ブロックからカード画像を生成します。

[FGO カード生成を開く](./CardGenerator.md)

## データソース

内蔵 FGO 画像索引を使い、利用可能な場合は Atlas Academy JP 参照データも使用します。
