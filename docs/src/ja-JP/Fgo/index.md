# Fate/Grand Order ツール

FGOサーヴァントポートレート抽出と処理のための総合ツールキット。

## 利用可能なツール

### キャラクター検索
精密な選択コントロールとアスペクト比プリセットを備えたサーヴァントポートレート抽出ツール。

[キャラクター検索を開く](./CharaFinder.md)

## 機能

- 高解像度サーヴァントポートレート抽出
- カスタムアスペクト比選択（1:1、4:3、3:2、16:9、カスタム）
- ピクセルレベルの精密選択コントロール
- 背景色カスタマイズ
- バッチ処理サポート
- 自動クロッピングのための顔検出
- 多言語サーヴァントデータベース

## 技術詳細

このツールはAtlas Academy APIと統合し、すべてのFGOサーバー（NA、JP、CN）で正確なサーヴァントデータを提供し、さまざまな用途での効率的なポートレート抽出を可能にします。
