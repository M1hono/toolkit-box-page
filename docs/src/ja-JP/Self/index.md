# Self ツール

個人用の小さなワークフローツールを置くためのモジュール入口です。

## ツール

### Mod Poster Studio

汎用画像編集と Mod 公開用カバーテンプレートの作業台です。

[Mod Poster Studio を開く](./PosterStudio.md)

## ルール

- ツールページは `Self` ディレクトリ内に置きます。
- 各ツールはゲーム系モジュールと結合せず、独立したページとコンポーネント境界を保ちます。
- この入口ページにはツールへのリンクと短い説明だけを置きます。
