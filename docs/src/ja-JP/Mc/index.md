# Minecraft ツール

Minecraftリソース処理と翻訳のための専門ツールキット。

## 利用可能なツール

### JSON 翻訳ツール
Minecraft言語ファイルとリソースパック用の高度なJSONファイル翻訳ツール。

[JSON 翻訳ツールを開く](./JsonTranslator.md)

## 機能

- JSON構造の保持
- バッチ翻訳サポート
- 多言語リソースパック生成
- 構文検証
- キーと値のペア編集
- 複数フォーマットへのエクスポート

## 使用例

- Minecraftモッドの翻訳
- 多言語リソースパックの作成
- 言語ファイルの管理
- JSON形式間の変換
