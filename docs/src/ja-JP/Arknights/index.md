# アークナイツツール

アークナイツゲームリソース抽出と分析のためのプロフェッショナルツールキット。

## 利用可能なツール

### キャラクター検索
マルチバリアントサポートと画像クロッピング機能を備えた高度なキャラクターポートレート抽出ツール。

[キャラクター検索を開く](./CharaFinder.md)

### ストーリートラッカー
名前ベースの検索とストーリーリーダー統合により、ゲーム全体でキャラクターのストーリー出演を追跡します。

[ストーリートラッカーを開く](./StoryTracker.md)

## 機能

- 多言語サポート（中国語、英語、日本語）
- キャラクターバリアント検出と管理
- ストーリー出演追跡
- 外部ストーリーリーダーとの統合
- バッチ処理機能
- 自動ポートレートクロッピングのための顔検出

## 技術詳細

このツールキットは公式アークナイツゲームリポジトリからデータを処理し、リソース抽出と分析のための直感的なインターフェースを提供します。
