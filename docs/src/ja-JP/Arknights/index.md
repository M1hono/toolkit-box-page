# アークナイツツール

アークナイツ素材検索とストーリー追跡のページ一覧です。

## ページ

### オペレーター立ち絵検索
オペレーター記録を検索し、立ち絵バリアントを選択し、画像を切り抜いて出力します。

[オペレーター立ち絵検索を開く](./CharaFinder.md)

### ストーリートラッカー
名前でストーリー出場記録を検索し、関連する閲覧先を開きます。

[ストーリートラッカーを開く](./StoryTracker.md)

## データソース

このプロジェクトに同梱されたアークナイツ立ち絵データとストーリー検索索引を使用します。
